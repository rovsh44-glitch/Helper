# Code of Conduct

## Scope

This repository is a public showcase surface for HELPER. We expect respectful, professional behavior in issues, pull requests, and related discussions.

## Expected Behavior

- be respectful and technically specific
- critique ideas, code, and claims without attacking people
- keep public reports non-confidential
- assume good faith, but challenge weak evidence directly
- follow repository boundaries and honest-status constraints

## Unacceptable Behavior

- harassment, abuse, or personal attacks
- discrimination or hateful conduct
- spam, trolling, or deliberate disruption
- posting credentials, private data, or confidential diligence material
- misrepresenting the repository as claiming human-level parity when it does not

## Reporting

For general conduct concerns, open an issue only if the matter is safe to discuss publicly. For sensitive concerns, use the contact path in [CONTACT.md](CONTACT.md). For security-sensitive matters, use [SECURITY.md](SECURITY.md).

## Enforcement

The maintainer may edit, hide, lock, or remove content that violates this code of conduct, and may restrict further participation when necessary.
