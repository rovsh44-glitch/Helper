# Certification Hub

## Purpose

This directory is the human-readable entry point for HELPER certification planning and operational review.

Use this folder when you need:
1. the active certification pack;
2. historical certification plans that still matter for traceability;
3. certification reference runbooks;
4. a single place that explains where runtime evidence is stored.

## Structure

1. `active/`
   Current normative certification documents.
2. `operations/`
   Execution runbooks and review checklists.
3. `history/`
   Historical certification plans retained for traceability.
4. `reference/`
   Supporting runbooks and reference certification examples.
