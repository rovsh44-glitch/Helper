# HELPER Model Experiment Notes

Этот каталог предназначен для `research-only` notes по model-side experiments.

Правила:

1. Notes в этом каталоге не изменяют product execution plan автоматически.
2. Любой experiment note должен ссылаться на `theme_id` из `doc/research/MODEL_EXPERIMENT_TRACK_REGISTRY.json`.
3. Notes здесь не являются release, parity или certification evidence.
4. Promotion возможен только через отдельный RFC и обновление authoritative execution docs.
