# Contact

This repository does not publish a direct public email address yet.

Use the prepared GitHub issue forms when this repository or a sanitized mirror is published:

- `Request demo or contact`
- `Submit a web-research prompt`
- `Apply as blind reviewer`

Current rules:

- Do not put secrets, credentials, private URLs, or personal data into public issues.
- Do not use public issues for vulnerability details. Use the process in `SECURITY.md`.
- Do not assume a public issue creates a private communication channel automatically.

Commercial and partnership intent:

- demos
- pilot conversations
- acquisition or investment outreach
- technical due-diligence interest

If a private contact route is not published yet, use the public request form only for initial routing and move sensitive discussion to a private channel after maintainer response.
