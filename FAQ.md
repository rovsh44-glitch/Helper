# FAQ

## What is HELPER?

HELPER is a local-first product shell that combines a React UI, an API host, and a backend runtime for research, planning, generation, and operator-guided execution.

## Is human-level parity achieved?

No. The current repository does not honestly support that claim. Canonical parity evidence is still incomplete.

## Is blind human evaluation complete?

No. Tooling exists, but the current authoritative blind reviewer corpus is not complete yet.

## Is the `14-day` parity window complete?

No. The counted window has not started yet.

## Is this repository meant to be pushed public as-is?

Not by default. The recommended model is a private-core repository plus a curated or sanitized public showcase.

## Does this repository include a hosted product?

No hosted public SaaS is promised here. The repository is set up for local operation and operator-managed environments.

## Can I contribute prompts or apply as a reviewer?

Yes, once the prepared GitHub issue forms are published in a public showcase repository or mirror.
